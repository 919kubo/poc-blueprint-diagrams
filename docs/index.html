What is PoC Blueprint?

PoC Blueprint は「作る前に、図で合意し、失敗を減らす」ための設計思想です。
成果物はコードではなく、まず L1の全体像（1枚）。ここで説明・認識合わせができます。

Why Mermaid?

Mermaidは、非エンジニアでも読めて、エンジニアはそのまま実装に接続できます。
編集は .mmd、公開は SVG固定 にすると、壊れずに共有できます。

  2. 必要なら、個別の共有スペースで短期デモ
